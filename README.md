🛍️ Customer Shopping Behavior Analysis
🚀 Overview

Built an end-to-end data analytics pipeline to analyze ~3,900 retail transactions and uncover insights into customer behavior, purchasing patterns, and revenue drivers.

Focus: Converting raw data into actionable business insights using Python, SQL, and Power BI.

🎯 Problem Statement

A retail company aims to leverage customer shopping data to:

Identify purchasing trends
Improve customer engagement
Optimize marketing and product strategies
⚙️ Tech Stack
Languages: Python, SQL
Libraries: Pandas, NumPy
Database: PostgreSQL
Visualization: Power BI
Tools: Jupyter Notebook, GitHub
🧠 Key Contributions
🔹 Data Processing (Python)
Cleaned and transformed dataset (handled missing values, standardized schema)
Engineered features like age groups and purchase frequency
Optimized dataset for downstream SQL analysis
🔹 Backend Data Analysis (SQL)
Designed structured queries to simulate business scenarios
Performed:
Revenue analysis by demographics
Customer segmentation (New, Returning, Loyal)
Discount impact analysis
Subscription vs non-subscription behavior
🔹 Data Visualization (Power BI)
Built an interactive dashboard displaying:
Customer distribution (~3.9K users)
Revenue by category and age group
Sales trends and subscription insights
📊 Key Results & Insights
Identified loyal customers as the dominant segment (~80%+)
Found that discount users can still be high spenders
Observed higher spending in express shipping users
Determined young adults contribute highest revenue share
Highlighted top-performing and discount-dependent products
💡 Business Impact
Improved targeting strategy using customer segmentation
Enabled data-driven decisions for pricing and discount strategies
Provided insights to enhance customer retention and engagement
